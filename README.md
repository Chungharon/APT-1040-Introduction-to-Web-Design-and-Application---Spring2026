# APT 1040: Introduction to Web Design and Application

**Spring 2026**

---

## Course Overview

This course introduces the fundamental concepts and practices of web design and development. Students will learn to create, style, and script web pages using industry-standard technologies such as HTML, CSS, and JavaScript. The course also covers web standards, usability, accessibility, multimedia integration, and emerging web technologies, preparing students for advanced web development and application design.

---

## Learning Objectives

By the end of this course, you will be able to:

- Build semantic, well-structured web pages using HTML5
- Style and layout web content with CSS3, including responsive design techniques
- Add interactivity and dynamic behavior using JavaScript
- Apply web standards, usability principles, and accessibility guidelines
- Integrate multimedia elements effectively into web projects
- Understand and implement emerging web technologies and best practices
- Deploy and manage web applications using modern development tools
